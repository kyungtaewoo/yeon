## 1. 기능 개요

카카오톡 친구를 초대하여 궁합을 확인하는 소셜 기능과,
궁합의 깊이에 따른 프리미엄 계층 구조를 정의합니다.

### 궁합 3단계 체계

```
┌─────────────────────────────────────────────────┐
│  1단계: 일반 궁합 (社會宮合) — 무료              │
│  사회적 관계 호환성 (직장, 친구, 동업 등)        │
│  분석 기준: 연주 합충, 월주 조화, 오행 상생상극   │
├─────────────────────────────────────────────────┤
│  2단계: 연인 궁합 (戀人宮合) — 프리미엄          │
│  연애/결혼 호환성                                │
│  분석 기준: 일간 합, 일지 육합, 관성, 재성 배치   │
├─────────────────────────────────────────────────┤
│  3단계: 깊은 궁합 (深層宮合) — 프리미엄+         │
│  내면·감정·본능적 호환성 (속궁합)                │
│  분석 기준: 일지 지장간, 도화살, 십이운성 목욕,   │
│  오행 과불급 보완, 무의식 욕망 패턴               │
└─────────────────────────────────────────────────┘
```

## 3. 궁합 분석 엔진 상세

### 3.1 일반 궁합 (社會宮合) — 무료

**분석 관점:** "이 사람과 같이 일하거나 친구로 지내면 잘 맞을까?"

```
분석 요소:
├── 연지(年支) 합충 관계                    (가중치 25%)
│   합(六合/三合) → +20점 / 충 → -15점
│   → 기본적인 성향 호환, "띠 궁합"에 해당
│
├── 월주(月柱) 조화                         (가중치 25%)
│   월간 합 → +10점 / 월지 합 → +10점
│   → 사회적 활동 패턴, 일하는 스타일 호환
│
├── 오행 상생상극 시너지                     (가중치 25%)
│   양쪽 오행 합산 균형도
│   상생 흐름 → +15점 / 상극 과다 → -10점
│   → 에너지적 보완 관계
│
├── 십성(十星) 사회적 역할 호환              (가중치 15%)
│   비겁+비겁 → 경쟁 관계 (+3)
│   식상+재성 → 생산적 관계 (+15)
│   관성+인성 → 안정적 관계 (+12)
│   → "같이 일하면 시너지 나는 조합인가?"
│
└── 원진살(元嗔殺) 체크                      (가중치 10%)
    子-未, 丑-午, 寅-酉, 卯-申, 辰-亥, 巳-戌
    원진살 존재 → -15점
    → 함께 있으면 다툼이 잦은 관계

총점: 0~100점 (기본 50점 + 가감)
```

**결과 표현 예시:**
```
"두 분은 서로 다른 강점을 가진 보완적 관계입니다.
 업무에서 함께하면 시너지가 높은 조합이에요.
 일반 궁합 점수: 78점 🤝"
```

### 3.2 연인 궁합 (戀人宮合) — 프리미엄 🔒

**분석 관점:** "연애/결혼 상대로서 얼마나 잘 맞을까?"

```
분석 요소:
├── 일간(日干) 합 — 가장 핵심 지표           (가중치 30%)
│   甲己合, 乙庚合, 丙辛合, 丁壬合, 戊癸合
│   합 → +30점 (천생연분급)
│   충 → -20점
│   상생 → +12점 / 상극 → -8점
│   → "이 사람에게 끌리는가?"
│
├── 일지(日支) 배우자궁 합충                  (가중치 25%)
│   육합 → +25점 / 삼합 → +15점
│   방합 → +10점 / 충 → -20점
│   형 → -10점 / 해 → -5점
│   → "결혼 후 가정이 안정적인가?"
│
├── 관성/재성 배치 (배우자 지표)              (가중치 20%)
│   여: 상대 일간이 내 정관 → +20점, 편관 → +10점
│   남: 상대 일간이 내 정재 → +20점, 편재 → +10점
│   → 전통 명리학의 핵심 배우자 지표
│
├── 연주+월주 종합 합 관계                    (가중치 15%)
│   연간/연지/월간/월지 합 → 각 +5점
│   → 가문, 환경, 생활 패턴 조화
│
└── 도화살(桃花殺) 상호 관계                  (가중치 10%)
    일지 기준 도화: 子→酉, 丑→午, 寅→卯...
    상대 사주에 내 도화 → +10점
    → 본능적 끌림, 매력

총점: 0~100점

+ 결혼 적합도 별도 산출 (0~100)
+ 연애 스타일 호환도 별도 산출 (0~100)
```

**결과 표현 예시:**
```
"💕 두 분의 연인 궁합: 91점
 일간 합(甲己合)으로 만나면 자연스럽게 끌리는 사이입니다.
 결혼 적합도: 87점 — 가정을 꾸리면 안정적일 사주 조합.
 연애 스타일: 두 분 모두 감성적이며 상대를 배려하는 성향."
```

### 3.3 깊은 궁합 (深層宮合 / 속궁합) — 프리미엄 🔒

**분석 관점:** "겉으로 드러나지 않는 내면, 감정, 본능적 차원에서 얼마나 잘 맞을까?"

**명리학적 근거:**
- 천간 = 겉모습 (의식적 성향)
- 지지 = 속마음 (내면적 성향)  
- 지장간 = 무의식 (숨겨진 욕망과 감정 패턴)
- 속궁합은 일지(배우자궁)의 지장간끼리의 관계를 분석하는 것

```
분석 요소:

├── 일지(日支) 지장간 상호 관계               (가중치 30%)
│   
│   일지 속에는 2~3개의 숨겨진 천간(지장간)이 있음.
│   이 지장간끼리의 합/충/생극 관계가 "속궁합"의 핵심.
│   
│   예: 내 일지 寅(인) → 지장간 [戊, 丙, 甲]
│       상대 일지 亥(해) → 지장간 [戊, 甲, 壬]
│       → 지장간 甲-壬 = 편인 관계 (보호/의지)
│       → 지장간 丙-壬 = 편관 관계 (긴장/끌림)
│       → 寅亥 육합 + 지장간 교류 = 속궁합 매우 좋음
│   
│   지장간 합 → +25점
│   지장간 상생 → +15점
│   지장간 상극 → -10점
│
├── 십이운성 목욕(沐浴) 관계                  (가중치 20%)
│   
│   목욕은 "발산, 개방, 솔직한 감정 표현"을 의미.
│   양쪽 사주에서 일간의 십이운성이 목욕 위치에 있으면
│   감정 표현이 솔직하고 내면 교류가 활발.
│   
│   양쪽 모두 목욕 → +15점 (감정 교류 활발)
│   한쪽만 목욕 → +8점
│   양쪽 모두 묘(墓) → -5점 (감정 억제)
│
├── 도화살(桃花殺) 심화 분석                  (가중치 20%)
│   
│   도화살 = 매력, 끌림, 감성적 교류의 척도
│   일지 기준 도화와 상대 사주의 관계:
│   
│   상대 일지가 내 도화 → +20점 (강한 본능적 끌림)
│   상대 시지가 내 도화 → +10점
│   상대 사주에 도화 전무 → +0점
│   도화 + 합 = "운명적 끌림" (+5점 보너스)
│
├── 오행 과불급 내면 보완                     (가중치 15%)
│   
│   내 사주에서 부족한 오행을 상대의 지장간이 보완.
│   → 무의식적으로 "이 사람 옆에 있으면 편하다"는 느낌.
│   
│   보완도 높음 → +15점
│   보완도 보통 → +8점
│   보완도 낮음 → +0점
│
└── 음양 심층 밸런스                          (가중치 15%)
    
    일간 음양 + 일지 음양 + 지장간 정기 음양
    → 3단계 음양 조화도 분석
    
    완전 조화 → +15점
    부분 조화 → +8점
    불조화 → +0점

총점: 0~100점
```

**결과 표현 (중요: 품위 있는 어조)**

"속궁합"이라는 단어는 UI에 사용하지 않습니다.
대신 **"깊은 궁합"**, **"내면 궁합"**, **"深層宮合"**으로 표현합니다.

```
"🌙 깊은 궁합: 89점

 겉으로 보이는 것 너머, 두 분의 내면은 깊은 곳에서 연결되어 있습니다.

 ✦ 무의식 조화도: 92점
   일지 지장간이 서로 상생하는 관계로,
   말하지 않아도 통하는 편안함을 느낄 수 있는 조합입니다.

 ✦ 감정 교류도: 85점
   양쪽 모두 감정 표현에 솔직한 사주 구조로,
   깊은 대화와 정서적 교감이 자연스러운 관계입니다.

 ✦ 본능적 끌림: 91점
   상대의 일지가 나의 도화 위치에 해당하여,
   만나면 자연스럽게 끌리는 강한 인력이 있습니다.

 ✦ 내면 보완도: 88점
   나에게 부족한 水 기운을 상대가 채워주어,
   함께 있으면 정서적으로 안정감을 느끼는 조합입니다.

 💡 심층 조언:
   두 분은 겉궁합(78점)보다 속궁합(89점)이 더 높은 관계입니다.
   처음엔 서먹할 수 있지만, 알수록 깊이 빠지는 인연이에요."
```

## 8. 지장간 데이터 테이블

```typescript
// lib/saju/jigangjang.ts

// 지지별 지장간 (여기, 중기, 정기 순)
export const JIJANG_GAN: Record<EarthlyBranch, [HeavenlyStem, HeavenlyStem?, HeavenlyStem?]> = {
  '子': ['壬', undefined, '癸'],      // 자: 임(여기), 계(정기)
  '丑': ['癸', '辛', '己'],           // 축: 계, 신, 기
  '寅': ['戊', '丙', '甲'],           // 인: 무, 병, 갑
  '卯': ['甲', undefined, '乙'],      // 묘: 갑, 을
  '辰': ['乙', '癸', '戊'],           // 진: 을, 계, 무
  '巳': ['戊', '庚', '丙'],           // 사: 무, 경, 병
  '午': ['丙', '己', '丁'],           // 오: 병, 기, 정
  '未': ['丁', '乙', '己'],           // 미: 정, 을, 기
  '申': ['己', '壬', '庚'],           // 신: 기, 임, 경
  '酉': ['庚', undefined, '辛'],      // 유: 경, 신
  '戌': ['辛', '丁', '戊'],           // 술: 신, 정, 무
  '亥': ['戊', '甲', '壬'],           // 해: 무, 갑, 임
};

// 지장간 궁합 계산
export function analyzeJijangGanCompatibility(
  branchA: EarthlyBranch,   // 내 일지
  branchB: EarthlyBranch,   // 상대 일지
  dayStemA: HeavenlyStem,   // 내 일간
  dayStemB: HeavenlyStem    // 상대 일간
): DeepCompatibilityScore {
  const jjgA = JIJANG_GAN[branchA];
  const jjgB = JIJANG_GAN[branchB];
  
  let score = 50; // 기본점수
  const factors: string[] = [];
  
  // 지장간끼리의 합/충/생극 관계 분석
  for (const stemA of jjgA.filter(Boolean)) {
    for (const stemB of jjgB.filter(Boolean)) {
      // 천간 합 체크
      if (isTianGanHe(stemA!, stemB!)) {
        score += 15;
        factors.push(`지장간 합(${stemA}${stemB}合) — 무의식 차원의 끌림`);
      }
      // 상생 체크
      if (isShengRelation(stemA!, stemB!)) {
        score += 5;
        factors.push(`지장간 상생 — 내면적 보완 관계`);
      }
      // 상극 체크
      if (isKeRelation(stemA!, stemB!)) {
        score -= 5;
        factors.push(`지장간 상극 — 내면적 긴장 관계`);
      }
    }
  }
  
  // 지장간 정기가 상대 일간과 맺는 십성 관계
  const jjgA_zhengqi = jjgA[jjgA.length - 1]!;
  const tenGodRelation = getTenGod(dayStemB, jjgA_zhengqi);
  // 정관/정인 = 안정 +8, 편관/편인 = 자극 +5, 식신 = 편안 +7
  
  return { score: clamp(score, 0, 100), factors };
}
```
